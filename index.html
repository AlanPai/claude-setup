🔧 Claude Config Setup Tool

一鍵安裝你的 Claude Code 個人設定到任何 Windows 裝置

開始前先問一下 — 你有 GitHub 帳號嗎？
這個工具需要把你的 Claude 設定放在 GitHub repo，方便跨裝置同步。
申請 GitHub 帳號 5 分鐘
1 前往 GitHub 註冊頁
打開 github.com/signup（會開新分頁）
2 填 email
用常用的 email（之後 commit 會掛在這個 email 上；之後可以改成 noreply 隱私版本）
3 設密碼
至少 15 字或 8 字含數字+小寫。建議用密碼管理器產一組。
4 選 username
會變成你的 GitHub URL（github.com/你的名字）。取短好記的，之後改要遷移會麻煩。
5 通過機器人驗證 + 驗證 email
解完拼圖後會寄驗證信到你填的 email，打開信點連結完成驗證。
6 （強烈建議）開 2FA 兩步驟驗證
登入後 → Settings → Password and authentication → Two-factor authentication。用 Authenticator app（Google Authenticator / 1Password / Authy）。
7 建立你的 Claude 設定 repo
右上 "+" → New repository → 名稱例如 claude_skill → Private（建議）→ Create。
建好後 URL 長這樣：https://github.com/你的名字/claude_skill.git，就是等下表單要填的 Repo URL。
基本設定 必填
到 github.com/settings/emails 開啟 "Keep my email addresses private" 取得
說明：新電腦要 clone 哪個 repo 到 ~/.claude/？Private repo 建議用 HTTPS（Git Credential Manager 會處理登入）
Notion 交班功能 選填
從 notion.so/my-integrations 取得。不填的話腳本會跳過這步
⚠️ Token 會出現在網址裡，只建議自己用的書籤，不要分享給別人
🔗 你的專屬連結
💡 書籤存起來，下次換裝置直接開這個連結，欄位自動帶入
安裝指令
📖 使用方式
第一次使用：填入資料 → 按「產生專屬連結」→ 書籤存起來
每次換裝置：開書籤連結 → 按「產生安裝指令」→ 複製 → 貼到 PowerShell 或 Git Bash
給朋友用：把這個網頁連結傳給他，讓他填自己的資料產生自己的連結
