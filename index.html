🔧 Claude Config Setup Tool

一鍵安裝你的 Claude Code 個人設定到任何 Windows 裝置

基本設定 必填
到 github.com/settings/emails 開啟 "Keep my email addresses private" 取得
Private repo 建議用 HTTPS（Git Credential Manager 會處理登入）
Notion 交班功能 選填
從 notion.so/my-integrations 取得。不填的話腳本會跳過這步
🔗 你的專屬連結
💡 書籤存起來，下次換裝置直接開這個連結，欄位自動帶入
安裝指令
📖 使用方式
第一次使用：填入資料 → 按「產生專屬連結」→ 書籤存起來
每次換裝置：開書籤連結 → 按「產生安裝指令」→ 複製 → 貼到 PowerShell 或 Git Bash
給朋友用：把這個網頁連結傳給他，讓他填自己的資料產生自己的連結
